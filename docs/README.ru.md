## Разработка

Из корня репозитория:

```bash
pnpm docs:dev
```

Запускается на `http://localhost:3051`.

## Сборка

Из корня репозитория:

```bash
pnpm docs:build
```

## Линтинг

```bash
pnpm lint:docs
```
