# Documentation

Built with [Docus](https://docus.dev)

## Languages

- English (`en`)
- Russian (`ru`)

## Development

From the repository root:

```bash
pnpm docs:dev
```

Runs at `http://localhost:3051`.

## Build

From the repository root:

```bash
pnpm docs:build
```

## Lint

From the repository root:

```bash
pnpm lint:docs
```
