## Иерархия

Поддерживается **2 уровня** вложенности - родитель и потомок:

- **Еда**
  - Продукты
  - Рестораны
  - Кофе
- **Транспорт**
  - Общественный транспорт
  - Такси
  - Топливо

::callout{icon="i-lucide-info"}
Транзакции назначаются только конечным категориям. Родительские категории агрегируют статистику от дочерних.
::

## Создание категории

Перейдите в **Категории** → **Новая**. Заполните:

- **Название** - обязательно, должно быть уникальным в пределах родителя
- **Иконка** - выберите из галереи [Material Design Icons](https://icones.js.org/collection/mdi) с поиском. По умолчанию: случайная иконка
- **Цвет** - палитра цветов с предпросмотром иконки. По умолчанию: случайный цвет
- **Родитель** - необязательно, выберите из доступных корневых категорий

При выборе родителя цвет категории автоматически наследуется от родительской.

Настройки по умолчанию: **Недавняя** включена, **Избранная** выключена.

### Дополнительные настройки

Для категорий **без дочерних**:

| Настройка | Эффект |
|-----------|--------|
| Избранная | Показывается в [быстром выборе](/ru/guide/transactions#как-работает-быстрый-выбор-категорий) на форме транзакции |
| Недавняя | Показывается в списке [недавних категорий](/ru/guide/transactions#как-работает-быстрый-выбор-категорий) (макс. 16, сортировка по дате последней транзакции, без пересечения с избранными) |

Для категорий **с дочерними**:

| Настройка | Эффект |
|-----------|--------|
| Применить цвет к дочерним | Обновляет цвет всех дочерних категорий |

::callout{icon="i-lucide-info"}
Категория не может стать родительской, если к ней уже привязаны транзакции. Сначала переместите или удалите эти транзакции.
::

## Удаление

**Удаление категории** удаляет все её транзакции. Категории с дочерними нельзя удалить - сначала удалите дочерние.

## Специальные категории

Две встроенные категории всегда присутствуют и не могут быть изменены или удалены:

- **Перевод** (`transfer`) - назначается автоматически при переводе между кошельками. Подробнее в разделе [Переводы](/ru/guide/transfers)
- **Корректировка** (`adjustment`) - для корректировки баланса без влияния на статистику. Подробнее в разделе [Транзакции](/ru/guide/transactions)

## Детали категории

Нажмите на категорию, чтобы открыть страницу детали с полной статистикой. Отображаются те же вкладки, что и на главной (Общее, Расход, Доход, Раздельно), отфильтрованные по этой категории.

Для родительских категорий дочерние отображаются в сетке - нажмите на любую для перехода на её страницу.

Действия в меню справа вверху: **Редактировать** и **Удалить**. Специальная категория `transfer` не может быть изменена или удалена.

## Статистика

На странице статистики - разбивка расходов по категориям с графиками. Сразу видно, на что уходит больше всего.
