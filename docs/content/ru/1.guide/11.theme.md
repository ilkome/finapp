Настройте внешний вид Финапки - цвета, тёмную тему и скругление углов на свой вкус. Все изменения применяются мгновенно без перезагрузки страницы.

## Где найти настройки темы

Панель темы доступна из нескольких мест:

- **Страница настроек** - полное управление темой
- **Страница входа** - панель инструментов вверху
- **Боковая панель** (только десктоп) - быстрый доступ слева

## Цветовой режим

Три варианта:

- **Светлый** - яркий интерфейс для хорошо освещённых мест
- **Тёмный** - комфортный для глаз при слабом освещении
- **Системный** - следует настройке ОС и переключается вместе с ней

## Основной цвет

Акцентный цвет для кнопок, активных состояний, ссылок и интерактивных элементов. Доступно более 20 цветовых палитр: red, orange, amber, yellow, lime, green, emerald, teal, cyan, sky, blue, indigo, violet, purple, fuchsia, pink, rose.

Есть режим **чёрный как основной** для монохромного вида.

## Нейтральный цвет

Тон фонов, границ и второстепенного текста:

- **Slate** - холодный сине-серый
- **Gray** - чистый нейтральный серый
- **Zinc** - слегка тёплый серый
- **Neutral** - сбалансированный серый
- **Stone** - тёплый бежево-серый

Нейтральный цвет влияет на карточки, боковую панель, поля ввода и большинство фоновых поверхностей.

## Скругление углов

6 уровней скругления для карточек, кнопок, полей ввода и модальных окон - от острых квадратных углов до полностью округлённых.

## Виджеты дашборда

На странице Статистики можно включать и выключать отдельные виджеты. Настройки сохраняются для каждой вкладки отдельно (Общее, Расход, Доход).

Доступные виджеты:

- **График** - столбчатый или линейный график по периодам. Можно скрыть полностью.
- **Быстрый выбор даты** - слайдер для навигации между периодами свайпом. Виден только при включённом графике.
- **Линия среднего** - горизонтальная линия на графике, показывающая среднее значение.
- **Кошельки** - сводка балансов кошельков с настраиваемым количеством отображаемых кошельков.
- **Средние значения** - исторические средние за N прошлых периодов с настраиваемым количеством.
- **Разбивка по категориям** - список, круглые иконки или вертикальный график. Режимы можно комбинировать.
- **Транзакции** - список транзакций за выбранный период.

Подробнее о каждой опции - в разделе [Статистика](/ru/guide/statistics#модальное-окно-настроек).

## Сохранение

Настройки темы сохраняются в `localStorage` и применяются до первого рендера - никакого мерцания при загрузке и никаких сетевых запросов.
