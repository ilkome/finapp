## Обзор

Главный экран (Статистика) с подробной разбивкой доходов и расходов. Транзакции-[корректировки](/ru/guide/transactions#корректировки) исключены из всей статистики.

## Вкладки

- **Общее** - чистый результат (доход минус расход)
- **Расход** - разбивка расходов по категориям
- **Доход** - разбивка доходов по категориям
- **Раздельно** - расход и доход рядом (только десктоп)

### Вкладка Раздельно

Отображает расходы и доходы бок о бок в двух колонках. Каждая колонка показывает свою разбивку по категориям, графики и суммы. Удобно для сравнения структуры расходов и доходов за один период.

Доступна только на десктопе - на мобильных экранах недостаточно места для двух колонок.

## Период

**Пресеты:** сегодня, вчера, текущая неделя, текущий месяц, текущий год, прошлая неделя, прошлый месяц, прошлый год.

**Произвольный период:** откройте календарь и выберите начальную/конечную даты.

**Быстрый слайдер:** свайп влево/вправо для навигации между периодами (например, по месяцам). Можно отключить в настройках.

**Группировка:** группировка данных по дню, неделе, месяцу или году в выбранном периоде.

**Максимальный период:** показать все доступные данные сразу.

## Графики

Выбор между столбчатыми и линейными графиками. Графики можно скрыть полностью. Каждая вкладка статистики (общее/расход/доход) имеет свою конфигурацию графика.

### Вид графика

На десктопе выбор между **половинным** (компактный) и **полным** (широкий) размером графика. На мобильных график всегда полной ширины.

### Тип графика

Переключение между **столбчатым** и **линейным** графиком для каждой вкладки.

### Линия среднего

Включите «Показать среднее» в настройках графика - горизонтальная линия на уровне среднего значения по всем видимым интервалам. Это среднее в текущем диапазоне дат - отдельно от исторических средних (раздел ниже).

## Разбивка по категориям

### Режимы отображения

Три режима визуализации, которые можно комбинировать - включайте любые одновременно:

- **Список** - подробный вид. Каждая категория отображается строкой с иконкой, названием, суммой и процентной полосой. Удобен для детального анализа - сразу видно долю каждой категории.
- **Круглые иконки** - компактный вид. Небольшие карточки с иконкой категории и суммой в две строки. Подходит для быстрого обзора, когда важно окинуть взглядом все категории.
- **Вертикальный график** - столбчатый график, где высота столбца отражает сумму. Под каждым столбцом - иконка категории. Даёт наглядное визуальное сравнение.

Переключение между режимами доступно в [модальном окне настроек](#модальное-окно-настроек). Там же есть быстрые пресеты: **Минимальный**, **Альтернативный** и **Стандартный**.

### Дополнительные опции

- **Показывать пустые категории** - отображать категории с нулевой суммой
- **Группировать по родителю** - объединять дочерние категории под родительскими. Родительские категории можно раскрывать и сворачивать.
- **Фон элементов** - цветной фон на карточках категорий
- **Процентные полосы** - полосы прогресса в режиме списка

### Клик для фильтрации

Нажмите на любую категорию в разбивке, чтобы временно отфильтровать весь вид (график + список транзакций) по этой категории. Нажмите ещё раз, чтобы снять фильтр.

### Переключение типа

На вкладке Общее нажмите на сумму дохода или расхода, чтобы переключить отображение между чистым доходом, только доходами и только расходами.

## Фильтрация

Фильтрация статистики по конкретным **кошелькам** или **категориям**. Множественный выбор с индикаторами-чипами активных фильтров. Фильтры сохраняются между сессиями.

### Ссылки с фильтрами

Настройки фильтров и дат хранятся в URL-параметрах (`?filterWallets=...&filterCategories=...`). Скопируйте URL, чтобы поделиться или добавить в закладки конкретный отфильтрованный вид.

## Средние значения

Переключение отображения средних значений. Настройка количества периодов для расчёта. Показывает историческое среднее за N прошлых периодов (например, средний месячный расход за последние 6 месяцев).

## Модальное окно настроек

Откройте иконку настроек на странице статистики. Настройки разделены на секции:

### График

- **Показать график** - включить/выключить график целиком
- **Быстрый выбор даты** - слайдер для навигации между периодами свайпом (виден только при включённом графике)
- **Показать линию среднего** - горизонтальная линия на уровне среднего значения
- **Вид графика** - половинный (компактный) или полный (широкий). Только десктоп - на мобильном график всегда полной ширины.
- **Тип графика** - столбчатый или линейный

### Кошельки

- **Показать кошельки** - отображать сводку балансов кошельков
- **Количество** - сколько кошельков показывать (от 1 до общего числа)

### Средние значения

- **Показать средние** - отображать исторические средние за прошлые периоды
- **Количество периодов** - за сколько прошлых периодов рассчитывать среднее (например, 6 месяцев)

### Категории

Быстрые пресеты для настройки вида разбивки по категориям:

- **Минимальный** - только список без дополнительных элементов
- **Альтернативный** - список с цветным фоном
- **Стандартный** - список с процентными полосами и круглыми иконками

Расширенные переключатели: фон элементов, процентные полосы, круглые иконки, вертикальный график, список, транзакции.

::callout{icon="i-lucide-info"}
Каждая вкладка статистики (Общее, Расход, Доход) хранит свою конфигурацию независимо. Например, на вкладке Расход можно включить линейный график, а на Доход - столбчатый.
::

## Пропуск пустых интервалов

На страницах деталей категорий в режиме максимального периода используется «пропуск пустых» - интервалы без транзакций сворачиваются, показываются только периоды с активностью.
