## Вход через Google

Войдите с помощью Google-аккаунта, чтобы:

- Синхронизировать данные между устройствами
- Хранить резервную копию на сервере
- Открывать данные из любого браузера

### Как войти

1. Откройте [finapp.ilko.me](https://finapp.ilko.me) или запустите установленное приложение
2. Нажмите **Войти через Google**
3. Выберите аккаунт и подтвердите разрешения
4. Финапка загрузит кошельки, категории и транзакции

При первом входе Финапка проведёт по онбордингу: создайте первый кошелёк и категорию.

## Демо-режим

Не хотите регистрироваться? Попробуйте демо-режим с готовыми данными:

- 1000 случайных транзакций за 2 года
- 18 категорий с иерархией родитель-дочерний
- 6 кошельков в разных валютах (включая архивный)
- Все функции работают - создание, редактирование, удаление, статистика

Данные хранятся локально в браузере - ничего не отправляется на сервер.

Подробнее в разделе [Настройки](/ru/guide/settings#демо-режим).

## Синхронизация

После входа данные синхронизируются между всеми устройствами в реальном времени. Добавили транзакцию на телефоне - она тут же появится на компьютере.

**Выбор языка** тоже синхронизируется с сервером - смените язык на одном устройстве, и он восстановится на всех остальных.

## Офлайн-режим

Финапка работает без интернета. Вот как это устроено:

### Что происходит при потере соединения

- Приложение продолжает работать - данные загружаются из локального кеша
- Вы можете создавать, редактировать и удалять транзакции, кошельки и категории
- Все изменения сохраняются в очередь на устройстве

### Что происходит при восстановлении соединения

- Накопленные изменения автоматически отправляются на сервер
- Очередь обрабатывается в порядке: кошельки, категории, транзакции, настройки
- Если вы создали кошелёк офлайн, а затем добавили в него транзакцию - связи сохранятся корректно

### Ограничения

- Курсы валют не обновляются без интернета - используются последние закешированные
- Синхронизация между устройствами возобновляется только при подключении

## Выход из аккаунта

Перейдите в **Настройки** → **Выйти**. Выход удаляет все локальные данные с устройства. При следующем входе всё синхронизируется обратно с сервера.

::callout{icon="i-lucide-alert-triangle" color="warning"}
Выход на одном устройстве не затрагивает другие - данные надёжно хранятся на сервере.
::

## Удаление аккаунта

Для полного удаления аккаунта и всех данных перейдите в **Настройки** → **Удалить все данные**. Это безвозвратно удалит все кошельки, категории и транзакции с сервера. Действие требует подтверждения и не может быть отменено.

Подробнее в разделе [Настройки](/ru/guide/settings#удаление-данных).
