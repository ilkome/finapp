## Обзор

Кошельки представляют, где хранятся ваши деньги - банковские счета, наличные, кредитные карты, криптокошельки.

## Типы кошельков

- **Cash** - наличные
- **Cashless** - банковские счета и электронные платежи
- **Credit** - кредитные карты (с поддержкой кредитного лимита)
- **Deposit** - вклады и депозиты
- **Crypto** - криптовалюта
- **Debt** - займы и долги

## Создание кошелька

Перейдите в **Кошельки** → **Новый**. Заполните:

- **Название** - обязательно, отображается везде
- **Валюта** - открывает модальное окно с поиском по всем 165 валютам. Используемые валюты отображаются первыми, затем остальные. Поиск по коду или названию. По умолчанию: **USD**
- **Цвет** - открывает нижнюю шторку с палитрой цветов и произвольным вводом цвета. Показывает превью кошелька с выбранным цветом
- **Тип** - наличные (по умолчанию), безнал, кредит, крипта, вклады, долги
- **Описание** - необязательные заметки о кошельке

### Дополнительные настройки

| Настройка | Эффект |
|-----------|--------|
| Кредитный лимит | Только для типа **кредит**. Показывает долг, доступный кредит и лимит |
| Не считать в общем балансе | Баланс кошелька не учитывается в итого |
| Доступные для снятия | Отметить как счёт для снятия |
| Архивный | Скрыт из активных списков и исключён из всех итогов. Виден только во вкладке-фильтре «Архив» |

## Список кошельков

Просмотр всех кошельков с балансами.

### Группировка

Группировка по валюте, по типу или вложенная группировка (валюта + тип). Каждая группа может быть свёрнута/развёрнута отдельно. Кнопка «развернуть/свернуть все» управляет всеми группами сразу.

При вложенной группировке первичная группировка (например, по валюте) создаёт верхнеуровневые группы, а вторичная (например, по типу) - подгруппы внутри каждой.

### Вкладки-фильтры

Вкладки-фильтры вверху показывают суммы по типам кошельков в базовой валюте:

| Вкладка | Показывает |
|---------|-----------|
| Итого | Все неархивные кошельки |
| Cash / Cashless / Credit / Crypto / Deposit / Debt | Кошельки данного типа (архивные исключены) |
| Снятие | Все кошельки, отмеченные для снятия (архивные исключены) |
| Доступно | Сумма снятия минус долг по кредитным картам. Отображается только при наличии и кошельков для снятия, и кредитных кошельков |
| Архив | Только архивные кошельки |
| Исключены из итого | Кошельки, исключённые из общего баланса (архивные исключены) |

Нажмите на вкладку для фильтрации списка. Сумма по каждой вкладке отображается в чипе.

Архивные кошельки полностью исключены из всей статистики и фильтров - они отображаются только во вкладке **Архив**. При переключении валюты, если текущий фильтр становится пустым (например, в выбранной валюте нет архивных кошельков), фильтр автоматически сбрасывается на **Итого**.

### Порядок

Изменение порядка кошельков перетаскиванием (доступно через иконку меню).

## Детали кошелька

Нажмите на кошелёк, чтобы увидеть все транзакции со статистикой. Переключайтесь между вкладками Общее, Расход, Доход и Раздельно - как на [Статистике](/ru/guide/statistics). Фильтрация по периоду.

### Кредитная карта

Для кредитных кошельков с настроенным лимитом на странице отображаются три значения:

- **Долг** - текущий баланс (сколько должны)
- **Доступно** - кредитный лимит минус долг
- **Лимит** - настроенный кредитный лимит

### Действия

В меню справа вверху на странице кошелька: **Редактировать** открывает форму кошелька, **Удалить** удаляет кошелёк и все его транзакции. Диалог подтверждения показывает, сколько транзакций будет затронуто.

## Начальный баланс

Новые кошельки создаются с нулевым балансом. Чтобы задать начальный баланс, создайте транзакцию-**корректировку** с нужной суммой. Корректировки не влияют на статистику, поэтому отчёты остаются чистыми. Подробнее в разделе [Транзакции - Корректировки](/ru/guide/transactions#корректировки).

## Расчёт баланса

Баланс рассчитывается автоматически: доходы прибавляются, расходы вычитаются. Переводы корректируют оба кошелька.

## Мультивалютность

У каждого кошелька своя валюта. Курсы обновляются ежедневно с [Open Exchange Rates](https://openexchangerates.org) - все балансы конвертируются в базовую валюту для единого итога.
