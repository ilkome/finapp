## Базовая валюта

Все балансы конвертируются в базовую валюту по ежедневным курсам. По умолчанию - **USD**.

## Язык

- **Английский** (en)
- **Русский** (ru)

Язык определяется автоматически по настройкам браузера. Также доступен на странице входа.

Выбранный язык синхронизируется на сервер - он будет восстановлен на любом устройстве, где вы войдёте в аккаунт.

## Демо-режим

Хотите посмотреть, как всё работает? Демо-режим создаёт примерные данные - без регистрации.

- 1000 случайно сгенерированных транзакций за 2 года
- 18 категорий с иерархией родитель-дочерний
- 6 кошельков в разных валютах (включая архивные)
- Все функции работают (создание, редактирование, удаление, статистика)
- Данные хранятся локально в вашем браузере

Действия:
- **Обновить демо данные** - перегенерировать все тестовые данные
- **Выйти из демо режима** - вернуться на страницу входа

## Курсы валют

Курсы обновляются автоматически с [Open Exchange Rates](https://openexchangerates.org) раз в день.

### Страница валют

На странице **Валюты** (`/currencies`) отображаются все валюты с курсами относительно базовой валюты.

**Два режима отображения:**

- **Используемые** - только валюты из ваших кошельков. Компактный список для быстрого просмотра актуальных курсов.
- **Все валюты** - полный список из 165 валют с возможностью поиска по коду или названию.

**Действия:**

- Нажмите правой кнопкой на валюту → **Установить базовой** для смены базовой валюты
- Базовая валюта отмечена меткой в списке
- При смене базовой валюты все курсы пересчитываются относительно новой

**Кеширование:** курсы сохраняются локально - страница работает без интернета, используя последние загруженные данные.

## Тема

Настройка цветов, тёмной темы и скругления углов. Подробнее в разделе [Тема](/ru/guide/theme).

## Удаление данных

На странице настроек можно удалить все данные. Это безвозвратно удалит все кошельки, категории и транзакции с сервера. Требуется подтверждение. Нельзя отменить.
