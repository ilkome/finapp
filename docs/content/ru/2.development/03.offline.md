## Установка PWA

- **Десктоп** — значок установки в адресной строке
- **Мобильные** — «Добавить на главный экран» из меню браузера

Приложение открывается в автономном режиме (standalone) без интерфейса браузера.

## Офлайн

Все данные хранятся локально. Транзакции, кошельки и категории можно создавать, редактировать и удалять без интернета.

### Очередь операций

Изменения офлайн записываются в локальную очередь:

- Тип сущности (транзакция, кошелёк, категория)
- Тип операции (создание, обновление, удаление)
- Данные и временная метка

Очередь оптимизируется — создание и последующее обновление одного элемента объединяются в одну операцию.

### Синхронизация при восстановлении связи

Очередь обрабатывается автоматически. Операции отправляются на сервер по порядку, локальные ID заменяются серверными.

## Стратегия синхронизации

- **Транзакции** — дельта-синхронизация. Загружаются только изменения с момента последней синхронизации по временной метке.
- **Кошельки и категории** — полная синхронизация. Коллекции небольшие, загружаются целиком.

Локальный кеш — `localforage`.
