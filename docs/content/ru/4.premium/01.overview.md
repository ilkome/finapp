Finapp Premium - это расширение базового приложения через механизм **Nuxt Layers**. Он живёт в отдельном репозитории ([finapp-premium](https://github.com/ilkome/finapp-premium)) и использует открытое базовое приложение как Git-субмодуль.

## Что добавляет Premium

- **Telegram-бот** - добавляйте транзакции из мессенджера командами, свободным текстом или по фото чека. Подробнее в разделе [Telegram-бот](/ru/premium/telegram-bot).
- **AI-функции** - разбор свободного текста и распознавание фотографий чеков на основе LLM.

## Как это работает

Premium - это [Nuxt Layer](https://nuxt.com/docs/getting-started/layers), который располагается поверх базового приложения. Он добавляет новые компоненты и страницы, не дублируя общий код. Базовое приложение подключено как Git-субмодуль, указывающий на ветку `convex`.

```
finapp-premium/
├── base/          # Git субмодуль → ilkome/finapp (ветка convex)
├── app/           # Компоненты и страницы только для Premium
└── nuxt.config.ts # Расширяет базовый слой через FINAPP_BASE_PATH
```

Для деплоя и локальной разработки смотрите раздел [Деплой](/ru/development/deployment#premium-finapp-premium).
