## PWA Installation

Finapp is a Progressive Web App (PWA). You can install it on your device for a native app-like experience:

- **Desktop** — click the install icon in your browser's address bar
- **Mobile** — use "Add to Home Screen" from the browser menu

The installed app runs in standalone mode without browser UI.

## Working Offline

All your data is stored locally. You can create, edit, and delete transactions, wallets, and categories while offline.

### Offline Queue

When you make changes offline, they are saved to a local queue. Each operation records:

- The entity type (transaction, wallet, or category)
- The operation type (create, update, or delete)
- The data and timestamp

The queue automatically optimizes itself — for example, if you create and then update the same item offline, they merge into a single create operation.

### Sync on Reconnect

When connectivity is restored, the offline queue is processed automatically. Operations are sent to the server in order, and local IDs are remapped to server IDs upon success.

## Sync Strategy

Finapp uses different sync strategies depending on the data type:

- **Transactions** — delta sync. Only changed transactions since the last sync are fetched, using a sync timestamp.
- **Wallets and Categories** — full sync. The complete list is fetched each time, since these collections are typically small.

Data is cached locally using `localforage` for fast access between sessions.
