# @finapp/app

[English](./README.md) | **Русский**

Пакет Nuxt-приложения Finapp.

Установка, настройка окружения и команды разработки описаны в корневом README ([English](../README.md) | [Русский](../README.ru.md)):

- Dev-сервер приложения: `pnpm dev`
- Dev-бэкенд Convex: `pnpm dev:convex`
- Сборка приложения: `pnpm --filter @finapp/app build`
- Линтинг приложения: `pnpm lint:app`
